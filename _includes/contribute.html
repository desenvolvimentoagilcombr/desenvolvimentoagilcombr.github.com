Como contribuir

O DesenvolvimentoÁgil.com.br é um projeto Open Source feito com o apoio da ImproveIT com a HE:labs. Mudanças e adições tanto no conteúdo quando no layout são bem-vindas, desde que sigam o Guideline de contribuição para o projeto.

Começando

  • Crie uma conta no GitHub, caso não possua.
  • Crie uma issue com a sua alteração. Certifique-se de que não exista alguma issue igual a sua.
  • Descreva claramente a mudança que você deseja fazer.
  • Faça um Fork do repositório.

Fazendo as mudanças

  • Crie um branch a partir do master.
    • Para criar um branch `git checkout -b minha_contribuicao`
  • Faça seus commits.

Enviando as mudanças

  • Faça um Push de suas mudanças para o branch no repositório em que você fez o Fork.
  • Envie um Pull Request para o repositório na organização desenvolvimentoagilcombr.
